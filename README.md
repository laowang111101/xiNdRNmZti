# 简易版图书管理系统后台 java1219

## 项目概述

本项目为一套简易的图书管理系统后台前端项目，基于Vue2开发，拥有友好美观的用户界面及多项功能模块，适用于进行二次开发，形成各种类型的管理系统应用。

## 技术栈

### 前端框架
- Vue2
- Vue Router
- Vuex

### UI组件库
- Element UI：提供美观且易用的管理后台界面

### 数据模拟
- MockJS：内置MockJS模拟后端接口，便于无缝切换至真实后端服务

### 状态管理
- Vuex：进行状态管理并支持持久化存储

### 权限控制
- 基于角色的动态路由权限控制

### 主题设置
- 支持明暗主题切换

### 组件封装
- 高度封装的通用组件，例如分页、面包屑等

## 功能模块

### 仪表盘
- 静态仪表盘：展示系统核心数据，支持多样的数据可视化图表

### 图书管理
- 图书列表
- 图书添加/编辑
- 图书分类管理

### 会员管理
- 会员列表
- 会员添加/编辑
- 会员详情查看

## 数据处理

- 使用MockJS模拟后端接口，使得前端可以独立运行，无需依赖后端服务
- Axios封装HTTP请求，实现请求/响应拦截
- 数据持久化存储，维护登录状态

## 二次开发支持

- 可以对接真实的后端API，移除Mock数据
- 支持扩展更多业务功能，如借阅管理、预约系统、电子书阅读等
